# CB-FUNK-DSP-SQUELCH

-- Eine Active DSP Rauschsperre für CB-Funk --

Der Squelch am Funkgerät muss offen sein! Denn die Software sucht Live nach Sprache im Rauschen. 
Eine Umschaltung gibt es auch für AM/SSB. Hier wird aber nur nach Pegel gesucht.

Ein HTML geschriebenes Programm mit JavaScript und einer C++ zu WASM! Es kann mit einem Server laufen, aber auch als File!
Beim Chrome muss man schauen, weil sie stärkere Sicherheitsprotokolle hat. Unter Mozilla läuft es ohne Probleme.

Es müssen alle Files runtergeladen werden! Die index.html und squelch.js müssen mit der LICENSE.txt im gleichen Ordner liegen!

Viel Spass und Erfolg mit der Software!
